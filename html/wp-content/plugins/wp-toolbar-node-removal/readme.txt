== Description ==
This plugin is Add-On for [WP ToolBar Removal](http://wordpress.org/plugins/wp-toolbar-removal/). Completely Remove <code>Admin Bar Node</code>, Frontend, Backend, Free Memory, Speedup Dasboard, on WordPress 3.1+ 3.2+ 3.3+ (only), for Third Party (free) Plugins. Is based on [WP Admin Bar Removal Node](http://wordpress.org/plugins/wp-admin-bar-node-removal/)

Work under [GPLv2](http://www.gnu.org/licenses/gpl-2.0.html) or later License. Implement [GNU style](http://www.gnu.org/prep/standards/standards.html) coding standard indentation.
